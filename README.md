## 🚀 部署到 GitHub Pages

### 步骤 1：创建仓库
1. 登录 GitHub
2. 创建新仓库，例如 `novel-tongluowan`
3. 勾选 "Add a README file"

### 步骤 2：上传文件
1. 点击 "Add file" → "Upload files"
2. 把所有文件拖进去（index.html, css/, js/）
3. 点击 "Commit changes"

### 步骤 3：启用 Pages
1. 进入仓库 Settings
2. 左侧菜单找到 Pages
3. Source 选择 `main` 分支，文件夹选择 `/ (root)`
4. 点击 Save

### 步骤 4：访问
等待 1-2 分钟，访问：
```
https://你的用户名.github.io/仓库名/
```
